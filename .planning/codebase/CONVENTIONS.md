# Conventions

## Coding Standards
- **TypeScript:** Strict typing used across the application.
- **Styling:** TailwindCSS for styling. Use `cn` utility from `src/lib/utils.ts` to merge classes dynamically.
- **Components:** Functional React components.
- **Linting & Formatting:** ESLint 9 and Prettier enforced.

## File Naming
- PascalCase for React components (e.g., `Header.tsx`).
- kebab-case or camelCase for utilities and config files.
