# Requirements

## Functional Requirements
- **Landing Page:** Hero section, about me, featured projects, and contact call-to-action.
- **Projects Showcase:** A dedicated section or page listing past work with details and links.
- **Contact Form:** A functional contact mechanism.
- **Responsive Navigation:** Mobile-friendly header and footer.

## Non-Functional Requirements
- **Performance:** High Lighthouse scores.
- **Accessibility:** Ensure semantic HTML and proper ARIA labels.
- **Design:** Modern aesthetics with dark mode support, using `shadcn/ui`.
